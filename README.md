# 모바일프로그래밍(2025-1) 프로젝트 LyricsSync
LyricsSync는 엉터리로 번역된 가사를 보고 노래를 맞추는 2~8인 멀티플레이 게임입니다.

  &nbsp;번역 된 가사의 규칙은 다음과 같습니다.
  
  &nbsp;1.한국어 -> 영어
  
  &nbsp;2.영어 -> 한국어
  
  &nbsp;3.기타언어 -> 한국어
  
  &nbsp;4.옛날 구글번역기의 어색한 기계번역투로 번역한다.
